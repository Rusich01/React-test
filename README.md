React Технічний Тест

Цей технічний тест міститиме лише фронтенд-компонент. Ми очікуємо, що виконання тесту займе приблизно 3 години. Будь ласка, уважно прочитайте вимоги нижче, щоб зрозуміти, що саме нам потрібно.

Вимоги
1. Створити React-додаток з використанням ES6 або новішого

Використовувати функціональні компоненти.

2. Створити веб-сторінку, яка дозволяє користувачу створювати та керувати бронюваннями

На сторінці повинні бути реалізовані такі операції:

Створення бронювання (Create)

Перегляд бронювання (Read)

Оновлення бронювання (Update)

Видалення бронювання (Delete)

3. Глобальний стейт

Зберігайте стан бронювань у глобальному сховищі стану (будь-якому на ваш вибір).

4. Валідація та досвід користувача

Реалізуйте логіку, яка запобігає подвійним (перекриваним) бронюванням.

Перевіряйте коректність дата початку та дата завершення бронювання.

5. Адаптивний дизайн

Сторінки мають бути повністю адаптивними для десктопу та мобільних пристроїв.

Термінологія

Бронювання — це коли гість обирає дату початку та дату завершення й надсилає запит на резервування об’єкта.

Подання роботи

Коли завершите роботу:

Завантажте ваш проєкт у публічний GitHub репозиторій.

Надішліть нам посилання електронною поштою.

Будь ласка, виконайте технічний тест протягом наступних 10 днів.
Дякуємо за приділений час і бажаємо успіху!
